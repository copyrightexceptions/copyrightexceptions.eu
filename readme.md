# copyrightexceptions.eu

This project contains the code and data for the 2020 rebuild of the [copyrightexceptions.eu](https://copyrightexceptions.eu/) website. As of February 2021 the website architecture has been validated and we are currently in the process of adding data on the implementation of the CDSM and InfoSoc exceptions (for now the Marrakesh and Orphan Works directive exceptions are out of scope — they will be added at a later stage).

## Implementation Status

As of 19 April 2026, the following EU27 country implementations are still missing (marked `draft: true`), excluding the MKD, OWD, and info53o exceptions.

| Country | Missing implementations |
|---------|------------------------|
| **CY** | dsm3, dsm4, dsm5, dsm8, info53a *(5 total)* |
| **IT** | info53e |
| **LU** | dsm3 |
